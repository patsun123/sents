Scoring formula

How storefront sentiment scores are currently derived from raw post and comment signals.

Current model
score =
polarity
* LN(upvotes + 2)
* content_type_weight
* subreddit_weight

content_type_weight =
post => 1.35
comment => 1.0 + LEAST(LN(reply_count + 1), 1.5) * 0.1
`polarity` is `+1` for positive and `-1` for negative sentiment.
`LN(upvotes + 2)` keeps engagement useful without letting viral outliers dominate.
Posts get a stronger headline-level bonus. Comments get only a mild reply-count bump, capped at about `1.15x`.
Community weights currently favor focused communities and discount adversarial pockets.
