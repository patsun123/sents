Four alternate visual treatments
of the same dashboard.

Same data, same information architecture, four distinctive aesthetics. Click through each one at 1440px and 375px (resize your window, or use DevTools device toolbar) before picking a winner. No build step — just open the files.

↔ A and C now ship a one-click toggle in their own headers, so visitors can flip between Neon Punk and Brutalist without leaving the page.

A · NEON PUNK

Gaming HUD

Cyberpunk console vibes. Scanlines, angled clip-path panels, hot magenta + electric cyan. Reads like a game interface.

↗
B · EDITORIAL

FT Alphaville

Cream paper, Fraunces serif, each ticker as a story with a lede and pullquotes. Reads as analysis, not a dashboard.

↗
C · BRUTALIST

Vercel × Linear

Off-white graph paper, 2px black borders, Archivo Black headlines, Klein-blue accent. Crisp, modern, confident.

↗
D · TERMINAL

Bloomberg

Dense monospace tables, amber highlights, keyboard shortcuts everywhere, ASCII bars. Power-user maximum density.
